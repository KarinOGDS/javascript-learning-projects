# javascript-learning-projects

Collection of JavaScript projects completed while studying Full Stack Development.

## Projects

### 01 - Followers Counter

A simple application that simulates social media follower growth using JavaScript events and DOM manipulation.

### 02 - Product Survey

Feedback survey application with form validation and dynamic content rendering.

### 03 - Restaurant Menu

Menu application demonstrating arrays, iteration methods and DOM updates.

### 04 - Employee Management System

Employee management application using arrays of objects, map(), filter(), find() and reduce().

### 05 - Speed Typing Test

Typing speed application that measures words per minute using timers and JavaScript calculations.

### 06 - Book Management System

CRUD-style book management application featuring dynamic rendering and form handling.

## Technologies Used

* HTML5
* JavaScript
* DOM Manipulation
* Event Handling
* Arrays and Objects
* Modern Array Methods
